Explicación para cambiar la cadena de la conexión de la página web
-Debe ir a la carpeta App_Data 
-debe encontrar la base de datos con nombre prueba4.mdf y hacerle doble clic
-se le abrirá la siguiente ventana o pestaña Deberá copiar la cadena de conexión que tendrá usted en ese momento.
-después se va a la carpeta configuración le hace clic, luego se dirige a clase conexión.
-borre la cadena de conexión que ya estaba puesta y pegue la que acaba de copiar.
-ya con eso debería poder correr de forma correcta la página web.
